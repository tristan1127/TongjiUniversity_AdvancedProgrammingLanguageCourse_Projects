# 漢諾塔綜合演示程式

## 專案簡介

這是一個基於命令列介面的漢諾塔綜合演示程式，整合了多種漢諾塔求解方式，並提供了圖形化演示功能。程式採用選單方式進行操作，支援自動演示和手動遊戲兩種模式。

## 功能特色

### 核心功能

1. **基礎演示模式（選單項 1-4）**
   - 遞迴求解過程展示
   - 橫向陣列顯示
   - 縱向陣列顯示
   - 完整的移動過程記錄

2. **圖形化演示（選單項 5-9）**
   - **選單項 5**：繪製三根圓柱底座
   - **選單項 6**：在指定圓柱上繪製圓盤（1-10 個）
   - **選單項 7**：演示第一個圓盤的移動過程
   - **選單項 8**：完整的漢諾塔自動求解演示
   - **選單項 9**：互動式遊戲模式（手動操作）

### 技術特點

- **偽圖形介面**：在 CMD 視窗中實現簡單的圖形顯示
- **動畫效果**：圓盤移動採用上移→平移→下移的流暢動畫
- **色彩豐富**：每個圓盤使用不同顏色區分
- **可調參數**：支援透過常數配置調整顯示效果

## 技術規格

### 開發環境

- **編譯器**：Visual Studio 2022
- **控制台**：Windows 控制台主機
- **字型設定**：新宋體 16 點陣
- **視窗大小**：120×40 字元以上
- **建議解析度**：1920×1080（FHD）

### 程式架構

專案包含以下檔案：

```
90-b1/
├── cmd_console_tools.h      # 控制台工具函式宣告
├── cmd_console_tools.cpp    # 控制台工具函式實現
├── cmd_hdc_tools.h          # 圖形工具函式宣告
├── cmd_hdc_tools.cpp        # 圖形工具函式實現
├── hanoi.h                  # 專案主標頭檔
├── hanoi_const_value.h      # 常數定義（可配置）
├── hanoi_menu.cpp           # 選單顯示與選擇
├── hanoi_multiple_solutions.cpp  # 各項演示功能實現
└── hanoi_main.cpp           # 主程式入口
```

### 程式設計約束

1. **遞迴函式限制**：整個程式僅使用一個遞迴函式（不超過 15 行）
2. **函式共用要求**：
   - 輸入參數必須共用一個函式
   - 橫向輸出必須共用一個函式
   - 縱向輸出必須共用一個函式
   - 繪製圓柱必須共用一個函式
   - 圓盤移動必須共用一個函式

3. **全域變數限制**：僅允許以下全域變數
   - 總移動步數：1 個變數
   - 圓柱上圓盤編號：3 個一維陣列或 1 個二維陣列
   - 圓柱上圓盤數量：3 個變數或 1 個一維陣列
   - 延時設定：1 個變數

4. **程式碼規範**：
   - 不允許使用指標、參考、結構體、類別
   - 不允許使用 `goto` 語句
   - 不允許使用全域變數（除上述允許項目外）
   - 不允許使用 C++ 的 `string` 變數
   - 不允許使用 `scanf`/`printf`
   - 達到「0 errors, 0 warnings」標準

## 功能展示

### 圖形化演示特點

- **圓柱繪製**：使用字元繪製三根圓柱及底座
- **圓盤顯示**：不同大小的圓盤以不同顏色顯示
- **移動動畫**：圓盤移動過程分為三個階段
  1. 上移至頂部
  2. 水平移動至目標圓柱
  3. 下移至目標位置
- **延時控制**：可調整動畫速度（0-200ms）

### 互動式遊戲（選單項 9）

- 透過鍵盤輸入控制圓盤移動（輸入兩個字母，如 AB 表示從 A 柱移到 B 柱）
- 自動檢測移動合法性（防止大盤壓小盤、空柱移動等錯誤）
- 記錄移動步數
- 完成時顯示遊戲結束提示

## 可配置參數

透過 `hanoi_const_value.h` 可調整以下參數：

### 圖形參數
- `HDC_Start_X/Y`：A 柱左上角座標
- `HDC_Base_Width`：立柱寬度
- `HDC_Base_High`：圓盤/底座高度
- `HDC_Top_Y`：圓盤上移的最高位置
- `HDC_Underpan_Distance`：立柱間距
- `HDC_Step_X/Y`：圓盤移動步進距離

### 顏色配置
- 背景底色
- 10 個圓盤的顏色（由小到大）
- 底座顏色（底盤+立柱）

### 文字顯示參數
- 各選單項的顯示起始座標
- 立柱字母相對偏移量
- 立柱之間的字元間距

## 使用說明

1. 編譯並執行程式
2. 根據選單提示選擇功能（1-9）
3. 按照提示輸入相關參數（圓柱編號、圓盤數量等）
4. 觀看演示或進行遊戲操作
5. 選擇 0 退出程式

## 注意事項

- 請確保使用 Windows 控制台主機（非 PowerShell）
- 建議在 1920×1080 解析度下執行以獲得最佳顯示效果
- 程式僅在螢幕上進行局部更新，不會整屏重繪（避免閃爍）
- 支援三種顯示版本：正常版、胖版、瘦版（透過不同的常數配置實現）
